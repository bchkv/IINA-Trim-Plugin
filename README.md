# README

Extract clips from videos. Functionally equivalent to QuickTime Trim or VLC Record.

## Installation

### 1. Install ffmpeg 

`brew install ffmpeg`

### 2. Install the plugin

Open **IINA → Plugin → Manage Plugins → Install from GitHub...**

Enter:

`https://github.com/bchkv/IINA-Trim-Plugin`

Then accept permissions.

## Usage

OPTION + S to mark the start and end points. Some settings are available in the settings menu.
